Speed, source links and Saucelabs

October 08 2013 Posted by Stuart Knightley

As one of the most commonly used components it’s important that the Repetition is fast, and to this end some unnesesary bindings have been removed. Montage has also been updated to use last week’s releases of FRB and Collections.

The List’s scrolling behaviour was fixed in both Matte and Digit, and a vertical slider was also added Digit.

Our API documentation has been updated to contain links to the source of symbol definitions, and the new features in FRB are now documented.

The montage-testing package recieved an update to Jasmine so that tests can be run on Saucelabs.

Releases

Digit v0.3.2 and minidom v0.1.0
